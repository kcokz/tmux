# tmux setup
## Reasons to setup
- Copy terminal text by mouse selection
- Copy to clipboard
## Steps to setup
### Install tmux-yank
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
### overwrite ~/tmux.conf
```
curl https://raw.githubusercontent.com/kcokz/tmux/main/tmux.conf --output ~/tmux.conf
```
